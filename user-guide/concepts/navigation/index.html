    Navigation

    To navigate the timeline in a Komposition project, you modify the focus. The focus describes the current part of the timeline you're focusing. It is modified using movement commands, such as the "Focus Left" and "Focus Up" commands.

    Visually, it's represented with a white border around the focused timeline part. For example:

    Key Bindings

    Navigation is mean to be keyboard-driven, in that you use your arrow keys or the hjkl keys to move around, like in Vim. You can also click any clip or gap in a parallel to move directly to it.

    Note

    There's currently no way to focus sequences or parallels using the mouse, but this should be addressed in the future.
